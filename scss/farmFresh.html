Farm Fresh App Info

Responsibilities

The Farm Fresh App is much like the Instacart App except that it enables the user to shop from local farmers. As one of the Front-End Developers, I collaborated with a cross-functional team of Developers. My responsibility was to build the user interface, register, and login forms using React, React Router, Hooks, and Formik (for validation).
